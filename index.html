Features - Responsive Design
  • 50% of web traffic is through mobile devices
  • How do you make your website look consistent on devices of different sizes?
  • This is the challenge of responsive design
  • Responsive design is an approach to building a web page so it looks consistent across different screen sizes
  • Without responsive design, a website might look fine on a desktop, but disorganized/jumbled on a mobile device

Features - Responsive Design
- Bootstrap allows developers to create responsive websites easily
- Provides a grid system where content on a page is separated into columns specified by an HTML class
Features - Responsive Design
- The columns in the grid collapse on top of one another based on “breakpoints” (implemented using media queries)
- Very easy to set up. Just need to add Bootstraps column classes to your HTML
Features - Responsive Design
- Bootstrap enables developers to quickly develop websites through out of the box HTML and CSS based styles and templates
- Templates for:
- Typography
- Forms
- Buttons
- Navigation bars
- Many of the templates have optional JavaScript plugins for additional functionality
Features - Responsive Design
Beauty of Bootstrap
- Minimal changes to the HTML are needed to transform the form into something elegant and responsive
- The only thing needed are to add Bootstrap classes to the HTML elements

Pros of using Bootstrap
- Very easy setup
- Offers a grid system that allows for responsive design
- Provides a variety of elegant styles and visual templates/components
- Consistent framework with support for all browsers -> reduces the amount of cross browser bugs
- Great documentation and community support -> easy to find solutions to problems
Cons of Bootstrap
- Widespread popularity of Bootstrap means that many websites look similar and have a “Bootstrap” look to them
- If you do add your own customization, may require a lot of style overriding making code harder to maintain
- Restricts possibilities of custom designs
- Namespace pollution of HTML classes from Bootstrap classes
Bootstrap Alternatives - Semantic UI
- Both offer out of the box styles and templates
- Bootstrap has very simple file structure and you can selectively download only the parts you need reducing file size
- Semantic UI has a more complicated file structure and has a larger file size
- Bootstrap only offers 1 default theme whereas Semantic UI has 20+ themes
- Bootstrap does not require JavaScript knowledge to use whereas Semantic UI does
Bootstrap Alternatives - Foundation
- Both offer a responsive grid system
- Both offer out of the box styles and templates
- Bootstrap has better browser compatibility
- Customizability is easier with Foundation
- Foundation is more novel -> Your website will look more unique
- Bootstrap being more popular has more community support
