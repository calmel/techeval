Features - Responsive Design
  • 50% of web traffic is through mobile devices
  • How do you make your website look consistent on devices of different sizes?
  • This is the challenge of responsive design
  • Responsive design is an approach to building a web page so it looks consistent across different screen sizes
  • Without responsive design, a website might look fine on a desktop, but disorganized/jumbled on a mobile device

Features - Responsive Design
- Bootstrap allows developers to create responsive websites easily
- Provides a grid system where content on a page is separated into columns specified by an HTML class
Features - Responsive Design
- The columns in the grid collapse on top of one another based on “breakpoints” (implemented using media queries)
- Very easy to set up. Just need to add Bootstraps column classes to your HTML
Features - Responsive Design
- Bootstrap enables developers to quickly develop websites through out of the box HTML and CSS based styles and templates
- Templates for:
- Typography
- Forms
- Buttons
- Navigation bars
- Many of the templates have optional JavaScript plugins for additional functionality

Beauty of Bootstrap
- Minimal changes to the HTML are needed to transform the form into something elegant and responsive
- The only thing needed are to add Bootstrap classes to the HTML elements
